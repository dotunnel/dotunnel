在 Windows设备上使用 ShadowsocksR

Windows

系统要求： Windows XP 及以上
软件版本： ShadowsocksR 4.8.0
设备要求： Windows PC / Tablet

1. 下载好应用后，获取订阅地址

如图中所示点击ssr订阅地址，自动复制订阅地址。


3. 完成订阅设置

打开 ShadowsocksR 程序，然后右键点击右下角的小飞机图标程序鼠标选择「服务器订阅」中「SSR服务器订阅设置」。重要提示：请先关闭服务器负载均衡


再弹出的窗口中「网址」一栏，粘贴您刚刚复制的节点订阅地址，并点击确定。


右键点击右下角的小飞机图标程序鼠标选择「服务器订阅」中「更新SSR服务器订阅（不通过代理）」。


右击小飞机图标，鼠标点击「服务器」中新的服务器节点信息，随意点击一条接入点。（重要！请务必操作此步骤看，否则后续步骤无法执行）


右击小飞机图标，鼠标点击「PAC」栏目下「更新PAC为GFWList」选项，等待执行成功。


最后右击小飞机图标，鼠标点击「系统代理模式」栏目下「PAC模式」选项，完成所有设置。
